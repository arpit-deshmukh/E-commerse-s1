# MERN E-commerce Project (Electronics)

This is a basic e-commerce application built using the MERN stack.

## Features

- User registration and login
- Product listing
- Add to cart
- Place order

## Tech Stack

- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Project Structure

/client - React frontend
/server - Express backend

## Setup Instructions

### Backend

cd server
npm install
npm run dev

### Frontend

cd client
npm install
npm start

## API Endpoints

- POST /api/auth/register
- POST /api/auth/login
- GET /api/products
- POST /api/products
- GET /api/cart
- POST /api/cart/add
- DELETE /api/cart/:index
- POST /api/orders
- GET /api/orders

## Note

This project intentionally contains incomplete features and design issues for learning and open-source contributions.
